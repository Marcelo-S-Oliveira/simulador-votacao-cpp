# simulador-votacao-cpp
Este repositório contém um código em C++ que implementa uma simulação básica de uma urna eletrônica. Permite o registro de votos para dois candidatos (A e B), contabiliza votos nulos e declara o vencedor ao final da votação.

Este é um programa simples em C++ que simula uma urna eletrônica para votação entre dois candidatos (A e B). Ele permite que os usuários registrem seus votos, contabiliza votos nulos e determina o vencedor com base na maioria dos votos.

## Funcionalidades

* Permite votar no Candidato A (digitando `1`).
* Permite votar no Candidato B (digitando `2`).
* Permite finalizar a votação (digitando `3`).
* Detecta e contabiliza votos inválidos como nulos.
* Exibe o total de votos para cada candidato e o número de votos nulos ao final da votação.
* Declara o vencedor ou indica um empate.
* Utiliza a localização para exibir mensagens em português.
* Realiza tratamento básico de entrada para garantir que apenas números sejam digitados para os votos.
